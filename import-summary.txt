Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

In xwalk_core_library:
* AndroidManifest.xml => xwalk_core_library\src\main\AndroidManifest.xml
* libs\armeabi-v7a\libxwalkcore.so => xwalk_core_library\src\main\jniLibs\armeabi-v7a\libxwalkcore.so
* libs\xwalk_core_library_java_app_part.jar => xwalk_core_library\libs\xwalk_core_library_java_app_part.jar
* libs\xwalk_core_library_java_library_part.jar => xwalk_core_library\libs\xwalk_core_library_java_library_part.jar
* res\ => xwalk_core_library\src\main\res\
* src\ => xwalk_core_library\src\main\java
* src\README.md => xwalk_core_library\src\main\resources\README.md
In DOMLauncher-CordovaLib:
* AndroidManifest.xml => DOMLauncherCordovaLib\src\main\AndroidManifest.xml
* src\ => DOMLauncherCordovaLib\src\main\java\
In DOMLauncher:
* AndroidManifest.xml => DOMLauncher\src\main\AndroidManifest.xml
* assets\ => DOMLauncher\src\main\assets\
* res\ => DOMLauncher\src\main\res\
* src\ => DOMLauncher\src\main\java\
